Here we give an example on training with PSPNet-ResNet50 with Cityscapes dataset.
You should put the pretrained models in ./initmodel and modify the config settings of corresponding yaml file in ./config/cityscapes.

Please find the dataset and pretrained models in https://mycuhk-my.sharepoint.com/:f:/g/personal/1155149725_link_cuhk_edu_hk/Em6TWGuPF01MrI3JmvC0zR8BDjiXq9XfwzH7zbC-DQz6pg?e=xbP6Ke

bash tool/train.sh cityscapes pspnet50
